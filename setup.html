Device Breakdown
💻
MacBook Pro
Streaming Hub
  • Runs OBS Studio for scene management
  • Handles encoding & stream output (RTMP)
  • Receives iPhone webcam via Continuity Camera
  • Receives gaming feed via HDMI capture card
  • Drives Monitor 1 over HDMI
📱
iPhone
Webcam Source
  • Used as high-quality webcam in OBS
  • Connected via Apple Continuity Camera (USB or wireless)
  • Appears as a camera source in OBS scene
🎮
PS5 / PC
Gaming Source
  • Primary gaming output (PS5 or Windows PC)
  • HDMI → Capture Card → MacBook (OBS)
  • Second HDMI → Monitor 2 (direct gameplay view)
🖥️
Monitor 1
Mac Display
  • Connected to MacBook via HDMI
  • Shows OBS, stream dashboard, chat
  • Primary working / streaming control monitor
🖥️
Monitor 2
Gaming Display
  • Connected directly to PS5 / PC via HDMI
  • Shows the game in full quality (no capture lag)
  • Primary view while gaming live
📺
Live Stream
Stream Output
  • OBS pushes RTMP stream to YouTube / Twitch
  • Combines webcam + game capture + audio
  • Encoded on MacBook before going out
