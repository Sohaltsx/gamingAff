# gamingAff

This site is ready for GitHub Pages deployment.

What is already set up:
- Static site files: `index.html`, `styles.css`
- Deployment workflow: `.github/workflows/deploy.yml`
- Pages-safe file: `.nojekyll`

To publish it:
1. Push this repository to GitHub.
2. In GitHub Settings → Pages, set Source to GitHub Actions.
3. Ensure the repository is public (GitHub Pages cannot publish private repositories on the free plan).
4. Merge or push to `main` to trigger the workflow.
